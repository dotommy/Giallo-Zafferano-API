API Script Guide

This script is designed to retrieve information about meals, pages, and recipes using specific API endpoints. Below are the details of the available endpoints:

List of Categories

  • 0: Antipasti
  • 1: Primi
  • 2: Secondi-piatti
  • 3: Dolci-e-Desserts
  • 4: Lievitati
  • 5: Piatti-Unici

Meal and Page Information

To retrieve information about a specific meal and page, use the following endpoint format:

/meal_{id_meal}/page_{id_page}

Here, {id_meal} refers to the index of the meal in the array meals_array, and {id_page} is the page number for the specified meal category. The script will return a JSON object with the following properties:

  • id: The unique identifier for the page.
  • title: The title of the page.
  • link: The link to the page.

Example:

/meal_2/page_1

This will return information about the first page of the "Secondi-piatti" category.

Recipe Information

If you want details about a specific recipe within a meal and page, use the following endpoint:

/meal_{id_meal}/page_{id_page}/recipe_{id_recipe}

Here, {id_meal} and {id_page} have the same meaning as before, and {id_recipe} is the identifier of the recipe obtained from the previous API call. The script will return detailed information about the specified recipe.

Example:

/meal_3/page_2/recipe_123

This will return information about the recipe with the ID 123 in the "Dolci-e-Desserts" category on the second page.
